# Florence Urban Land Use Dashboard

## Overview
This project analyses land-use patterns in Florence using OpenStreetMap data, QGIS, Excel, and Power BI.

## Objectives
- Identify major land-use categories
- Calculate land-use area distributions
- Create thematic maps and charts
- Support spatial analysis and urban planning insights

## Data Source
- OpenStreetMap
- QuickOSM
- QGIS

## Tools Used
- QGIS
- Excel
- Power BI
- OpenStreetMap

## Key Findings
- Forest was the dominant land-use category.
- Residential land represented the largest urban land-use type.
- Orchard and farmland occupied significant portions of the study area.
- Florence contains substantial green and agricultural spaces alongside urban development.
